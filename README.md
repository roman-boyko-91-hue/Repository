в рамках 2 курса Разработка на Python

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/roman-boyko-91-hue/Repository/tree/feature/homework_10_1
```
2. Установите зависимости:
```
poetry install
```

2. Запустите работу функций.
